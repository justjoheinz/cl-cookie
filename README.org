# CL-COOKIE

cookiecutter template for common lisp projects

The template creates

 - src directory
 - t directory
 - asd file
 - .gitignore

and executes

 - git init
 - qlot init
 - qlot install


Invoke with ~cookiecutter gh:justjoheinz/cl-cookie~.

Pull Request are very welcome.
